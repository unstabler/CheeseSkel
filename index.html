CheeseSkel

CheeseSkel은..

여러분의 시스템 환경을 치즈치즈하게 바꾸는 dotfiles / 스크립트 모음집입니다.

설치

0. 사용 중이신 배포판에서 제공하는 패키지 관리자로 필요한 패키지를 설치합니다.

sudo <pkgman> install git wget zsh vim

1. 저장소를 복제합니다.

git clone https://github.com/unstabler/CheeseSkel.git ~/.skel

2. 복제된 저장소 디렉토리로 이동합니다.

cd ~/.skel

3. 설치 스크립트를 실행합니다. \ ' ')/

perl installer.pl
